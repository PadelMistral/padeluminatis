🎯 Sistema Cuántico Ajustado - VICTORIA FIJA + VARIABLES INDIVIDUALES

Lógica del sistema:

  1. Nivel del partido: Se determina por la media de los equipos (reñido, normal, difícil, épico)
  2. Victoria/Derrota base: Puntuación fija según nivel del partido (7-16 pts victoria, -7 a -14 pts derrota)
  3. Multiplicador por diferencia: Basado en marcador real (6-0, 6-0 = 1.45x, partido ajustado = 1.05x)
  4. Variables individuales: Se suman todas y se dividen entre 2.5 (media ajustada)
  5. Puntuación final: Base × Multiplicador + Media de variables + Variación aleatoria (±0.4)

Tipos de partido según nivel:

  • 🏆 Reñido: Equipos similares (8.5 pts victoria, -14.2 pts derrota)
  • ⚔️ Normal: Diferencia moderada (7.2 pts victoria, -12.8 pts derrota)
  • 🔥 Difícil: Diferencia significativa (11.8 pts victoria, -10.5 pts derrota)
  • 💎 Épico: Diferencia muy grande (16.4 pts victoria, -7.3 pts derrota)

Variables individuales (suma/2.5):

  • 📊 Marcador (1.5-5.8 pts)
  • 🔥 Dificultad rivales (1.8-4.2 pts)
  • 👑 Nivel compañero (0.8-2.8 pts)
  • 🏅 Tipo partido (0.8-2.2 pts)
  • 🎯 Posición ranking (0.8-2.2 pts)
  • 🎓 Experiencia (0.5-1.8 pts)
  • 📈 Racha (0.5-1.5 pts)
  • 🎯 Consistencia (0.3-0.8 pts)
  • 💪 Intensidad (0.4-1.2 pts)
  • 🎯 Clutch (0.3-1.5 pts)

🎯 SISTEMA AJUSTADO: Victoria fija según nivel + variables individuales que diferencian a cada jugador. Partido reñido: ~8-12 pts, Partido normal: ~7-10 pts, Partido difícil: ~12-16 pts, Partido épico: ~16-22 pts.
Derrotas penalizan más: -7 a -18 pts según dificultad.
